Rules of Game 2048
  1. Game Grid: The game is played on a 4x4 grid, totaling 16 cells. Each cell can contain a tile with a numerical value.
  2. Initial Setup: At the start of the game, two tiles are randomly placed on the grid. These tiles can have a value of either 2 or 4.
  3. Tile Movement: The player can move tiles in four directions: up, down, left, or right. When the player chooses a direction, all tiles on the grid shift as far as possible in that direction until they reach the edge of the grid or collide with another tile.
  4. Merging Tiles: When two tiles with the same value collide while moving, they merge into a single tile. The value of the merged tile is the sum of the values of the two tiles that collided. For example, if two tiles with a value of 2 collide, they merge into a single tile with a value of 4.
  5. Adding New Tiles: After each move, a new tile with a value of either 2 or 4 appears on an empty spot on the grid. The placement of the new tile is random.
  6. Win Condition: The game is won when a tile with a value of 2048 is created. Once the player achieves this, they have successfully won the game.
  7. Game Over Condition: The game ends when there are no more empty spots on the grid and no possible moves can be made. This condition results in a game over.
